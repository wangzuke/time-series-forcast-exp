# 观测缺失条件下多变量时间序列预测实验计划

## 1. 实验背景

多变量时间序列预测是指利用多个相关变量的历史观测值预测未来一段时间内这些变量的变化。例如，在气象预测中，温度、湿度、气压和风速可以共同构成一个多变量时间序列；在交通预测中，不同路段的车流量也可以共同构成一个多变量时间序列。传统时间序列预测方法通常默认历史输入数据是完整的，即每个时间点、每个变量都有可用观测值。然而，在真实应用中，数据完整性往往难以保证。传感器故障、通信丢包、人工记录遗漏和设备离线等因素都会导致历史输入中出现观测缺失。

观测缺失会直接影响预测模型的输入质量。若缺失值被简单替换为均值、零值或前一个观测值，模型可能无法区分“真实观测值”和“人工填补值”，从而把低可信度信息误认为真实历史信号。已有研究指出，传统“先补值、再预测”的两阶段方法存在误差传播问题：补值阶段产生的错误会进入预测模型，并进一步影响未来预测结果。MissTSM 提出不显式填补缺失值，而是通过缺失特征感知注意力机制直接处理部分观测序列；CRIB 进一步指出，缺失位置没有真实监督值时，补值过程可能破坏数据分布并降低预测精度；CoIFNet 则提出将补值和预测放入同一框架中联合优化，以减少两阶段目标不一致带来的问题 [5]–[7]。

本实验聚焦的问题是：**当多变量时间序列的历史输入存在观测缺失时，不同缺失处理策略对未来预测性能的影响如何？** 实验不讨论多变量之间采样频率不同的异步采样问题，只研究“在已有统一时间网格上，部分时间—变量位置缺失”的情况。

## 2. 核心概念说明

为避免术语混淆，本实验涉及的关键概念定义如下。

| 概念 | 含义 |
|---|---|
| 多变量时间序列 | 同一时间段内同时记录多个变量的序列，例如温度、湿度、气压共同组成的气象序列。 |
| 观测缺失 | 在某个时间点、某个变量本应有观测值，但实际记录为空。 |
| 缺失掩码 | 一个与原始数据形状相同的 0/1 矩阵，用于标记哪些位置有观测、哪些位置缺失。1 表示有观测，0 表示缺失。 |
| 两阶段方法 | 先对缺失值进行填补，再把填补后的完整序列输入预测模型。 |
| 联合补值预测方法 | 在同一个模型中同时处理缺失值和未来预测任务。 |
| 不补值直接预测方法 | 不显式恢复缺失位置，而是让模型直接从部分观测序列中学习预测未来。 |
| 注意力机制 | 一类神经网络结构，用于让模型自动判断不同时间点或不同变量对预测目标的重要程度。 |
| Transformer | 一类基于注意力机制的深度学习模型，已广泛用于自然语言处理和时间序列预测任务。 |
| 均方误差 | 预测值与真实值差的平方平均值，用于衡量预测误差。 |
| 平均绝对误差 | 预测值与真实值差的绝对值平均值，用于衡量预测误差。 |
| 相对退化率 | 缺失输入条件下相对于完整输入条件的性能下降比例，用于衡量模型面对缺失时的鲁棒性。 |

## 3. 实验目标

本实验的总体目标是建立一个系统的观测缺失条件下多变量时间序列预测评估框架。具体目标包括：

1. 比较不同缺失率下各类方法的预测性能变化。
2. 比较随机点缺失、连续片段缺失和变量通道缺失三类缺失模式对预测性能的影响。
3. 检验传统“两阶段方法”是否存在误差传播问题。
4. 检验缺失掩码是否对预测模型有必要。
5. 比较简单填补后预测、强补值模型后预测、联合补值预测、不补值直接预测四类策略。
6. 建立可复现的观测缺失条件下多变量时间序列预测实验基准。

## 4. 研究问题与实验假设

| 编号 | 研究问题 | 实验假设 |
|---|---|---|
| 问题一 | 缺失率升高时，不同方法的预测误差如何变化？ | 缺失率越高，传统两阶段方法的误差增长越明显；直接利用部分观测序列的方法退化较慢。 |
| 问题二 | 随机点缺失和连续片段缺失对模型的影响是否相同？ | 连续片段缺失对预测的破坏更大，因为连续缺失会破坏局部时间变化模式。 |
| 问题三 | 补值效果好是否一定意味着预测效果好？ | 补值误差低不一定代表预测误差低，因为补值任务和预测任务的优化目标不同。 |
| 问题四 | 缺失掩码是否能够提升预测效果？ | 加入缺失掩码后，模型能够区分真实值与缺失位置，预测性能更稳定。 |
| 问题五 | 联合补值预测是否优于先补值再预测？ | 在高缺失率条件下，联合建模方法优于分离式两阶段方法。 |
| 问题六 | 简单线性模型在缺失场景下是否仍具有竞争力？ | 在部分规律性较强的数据集上，简单线性模型可能接近复杂模型，因此应作为必要基线。 |

## 5. 数据集

实验使用标准多变量长序列预测数据集。这些数据集在 PatchTST、iTransformer、DLinear 等时间序列预测研究中被广泛使用，因此具有较好的可比性 [1]–[3]。

| 数据集 | 数据领域 | 数据特点 | 实验作用 |
|---|---|---|---|
| ETTh1 | 电力变压器温度 | 变量数量适中，周期性明显 | 用于基础验证和参数选择 |
| ETTm1 | 电力变压器温度 | 时间粒度更细，序列更长 | 用于高频序列缺失实验 |
| Weather | 气象观测 | 多个物理变量相关性较强 | 用于检验跨变量依赖 |
| Electricity | 电力负荷 | 变量数量较多 | 用于高维多变量缺失实验 |
| Traffic | 交通流量 | 高维、周期性强 | 用于大规模多变量预测实验 |
| ExchangeRate | 汇率 | 变量较少、噪声较强 | 用于低维噪声序列实验 |

主实验使用 ETTh1、Weather、Electricity 和 Traffic 四个数据集。扩展实验使用 ETTm1 和 ExchangeRate。

所有数据按照时间顺序划分为训练集、验证集和测试集，不进行随机打乱。标准化仅使用训练集的均值和标准差，验证集和测试集不参与统计量计算，以避免未来信息泄露。

## 6. 缺失数据构造

实验只在历史输入窗口中注入缺失，预测目标保持完整。这样可以保证评估指标反映的是模型在“历史信息不完整”条件下的预测能力，而不是标签缺失造成的评估偏差。

设历史输入长度为 $L$，变量数为 $C$，原始输入为：

$$
X \in \mathbb{R}^{L \times C}
$$

缺失掩码为：

$$
M \in \{0,1\}^{L \times C}
$$

其中，$M_{t,c}=1$ 表示该位置有真实观测，$M_{t,c}=0$ 表示该位置缺失。模型实际接收的输入为：

$$
\tilde{X}_{1:L},\; M_{1:L},\; T_{1:L}
$$

其中，$\tilde{X}$ 表示缺失处理后的输入，$M$ 表示缺失掩码，$T$ 表示时间位置信息。

### 6.1 缺失类型

| 缺失类型 | 构造方式 | 对应真实场景 |
|---|---|---|
| 随机点缺失 | 在历史输入中随机选择单个时间—变量位置置为缺失 | 随机丢包、偶发记录失败 |
| 连续片段缺失 | 对某个变量连续若干时间步置为缺失 | 传感器短时故障、通信中断 |
| 变量通道缺失 | 对某些变量在整个输入窗口内置为缺失 | 某个传感器完全离线 |
| 混合缺失 | 同时包含随机点缺失、连续片段缺失和变量通道缺失 | 复杂真实缺失场景 |

CoIFNet 的实验明确覆盖随机点缺失和连续片段缺失，并报告在高缺失率下相对已有方法具有更强表现，因此本实验将这两类缺失作为主实验缺失模式 [7]。

### 6.2 缺失率设置

缺失率设为：

$$
r \in \{0.1,\, 0.3\}
$$

其含义如下：

| 缺失率 | 实验含义 |
|---|---|
| 10% | 轻度缺失 |
| 30% | 中度缺失 |

### 6.3 缺失生成规则

随机点缺失：对历史输入中的每个时间—变量位置独立随机决定是否缺失。

连续片段缺失：随机选择一个变量和一个起始时间，将该变量在连续 12、24 或 48 个时间步内的观测值置为缺失。

变量通道缺失：随机选择一部分变量，使这些变量在整个历史输入窗口内均不可见。

混合缺失：按照随机点缺失 50%、连续片段缺失 30%、变量通道缺失 20% 的比例混合生成。

## 7. 预测任务设计

实验采用多变量多步预测任务。模型输入为存在观测缺失的历史序列，输出为未来多个时间步的预测结果。

| 项目 | 设置 |
|---|---|
| 历史输入长度 | 96 和 336 |
| 未来预测长度 | 96、192、336、720 |
| 训练轮数 | 10 到 20 轮，使用早停策略 |
| 优化方法 | Adam 或 AdamW 优化器 |
| 学习率 | $1\times10^{-4}$、$5\times10^{-4}$、$1\times10^{-3}$ |
| 随机种子 | 2024、2025、2026 |
| 主评价指标 | 均方误差、平均绝对误差 |
| 辅助评价指标 | 相对退化率、鲁棒性斜率、训练时间、显存占用 |

## 8. 对比方法设计

实验比较四组方法：无缺失上界方法、简单填补后预测方法、强补值模型后预测方法、缺失感知预测方法。

### 8.1 无缺失基准测试

这一组方法在完整输入上训练和测试，用于获得无缺失条件下的预测性能。
我选择近几年较为著名，学术界反馈较好的几个模型进行测试验证
| 方法 | 说明 |
|---|---|
| PatchTST | 一种基于 Transformer 的时间序列预测模型，将时间序列切成片段，再用注意力机制建模片段之间的关系 [1]。 |
| iTransformer | 一种倒置 Transformer 预测模型，将不同变量作为建模对象，用注意力机制学习变量之间的关系 [2]。 |
| DLinear | 一种线性预测模型，但性能优异，用于检验复杂模型是否真的优于简单线性方法 [3]。 |



### 8.2 简单填补后预测方法

这一组方法先用简单规则填补缺失值，再输入预测模型。该部分回答的问题是：**面对观测缺失，最朴素的处理方式能够达到什么水平？**

为控制实验规模，本组只使用 **线性插值** 作为简单填补的代表方法。均值填补和前向填补在过往工作中已被广泛比较，其精度普遍弱于或与线性插值相近；保留线性插值作为代表已经足以体现“最简单两阶段方法”的水平。

| 方法 | 填补方式 | 预测模型 |
|---|---|---|
| 线性插值后预测 | 用缺失位置前后的观测值做线性插值 | DLinear、PatchTST、iTransformer |

DLinear 也纳入本组实验。其作用是判断：在相同简单填补策略下，复杂预测模型是否确实优于简单线性模型。若“线性插值 + DLinear”已经取得接近 PatchTST 或 iTransformer 的效果，则说明该数据集上的缺失预测问题可能主要由趋势或周期性驱动，而不一定需要复杂模型。

### 8.3 强补值模型后预测方法

这一组方法使用专门的时间序列补值模型先恢复缺失值，再进行预测。该部分回答的问题是：**使用强补值模型是否一定能够带来更好的预测结果？**

| 方法 | 说明 |
|---|---|
| SAITS 后接 DLinear | SAITS 先补齐缺失值，DLinear 再预测未来。 |
| SAITS 后接 PatchTST | SAITS 先补齐缺失值，PatchTST 再预测未来。 |
| SAITS 后接 iTransformer | SAITS 先补齐缺失值，iTransformer 再预测未来。 |

SAITS 的全称是 **Self-Attention-based Imputation for Time Series**，即“基于自注意力机制的时间序列补值方法”。它使用注意力机制同时建模时间依赖和变量关系，是时间序列补值任务中的典型强基线方法 [4]。

DLinear 纳入本组实验，用于判断强补值器的收益是否依赖复杂预测模型。如果“SAITS + DLinear”已经表现良好，说明性能提升可能主要来自补值阶段；如果“SAITS + PatchTST”或“SAITS + iTransformer”明显更好，则说明补值后的复杂时序建模仍然重要。

### 8.4 缺失感知预测方法

这一组方法不再简单地把缺失处理视为预测前的预处理步骤，而是在模型内部显式建模缺失。该部分回答的问题是：**直接建模缺失是否优于先补值再预测？**

| 方法 | 中文说明 | 处理思路 |
|---|---|---|
| MissTSM | 不补值时间序列建模方法 | 不显式恢复缺失值，直接通过缺失特征感知注意力机制利用部分观测序列 [5]。 |
| CRIB | 一致性正则信息瓶颈方法 | 直接从部分观测序列预测，通过约束表示学习过滤缺失引入的噪声 [6]。 |
| CoIFNet | 协同补值预测网络 | 将补值和预测放在同一框架中联合优化，输入包括观测值、缺失掩码和时间位置信息 [7]。 |

其中，CRIB 的全称是 **Consistency-Regularized Information Bottleneck**，可译为“一致性正则化信息瓶颈”。CoIFNet 的全称是 **Collaborative Imputation-Forecasting Network**，可译为“协同补值预测网络”。MissTSM 论文提出的核心思想是避免传统两阶段补值带来的误差传播；CRIB 论文提出直接从部分观测序列进行预测；CoIFNet 论文提出补值和预测目标统一优化 [5]–[7]。

## 9. 主实验设计

主实验用于回答：在不同缺失率和不同缺失模式下，各类方法的预测效果如何。

| 实验维度 | 设置 |
|---|---|
| 数据集 | ETTh1、Weather、Electricity、Traffic |
| 缺失类型 | 随机点缺失、连续片段缺失 |
| 缺失率 | 10%、30% |
| 历史输入长度 | 96 |
| 未来预测长度 | 96、336 |

| 重复次数 | 每个设置使用 3 个随机种子 |

主实验结果以均方误差和平均绝对误差为主，报告每个方法在不同数据集、不同缺失率、不同预测长度下的平均性能和标准差。

| 实验维度 | 设置 |

## 11. 消融实验设计

消融实验用于分析模型中关键组成部分的作用。

### 11.1 缺失掩码作用实验

该实验比较三种输入形式：

| 输入形式 | 说明 |
|---|---|
| 只输入填补后的数值 | 模型不知道哪些位置是真实观测，哪些位置是填补值。 |
| 输入数值和缺失掩码 | 模型可以区分真实观测和缺失位置。 |
| 输入数值、缺失掩码和时间位置 | 模型同时知道数值、缺失位置和时间顺序。 |

若第二种和第三种输入形式明显优于第一种，说明缺失掩码是必要信息。

### 11.2 补值误差传播实验

该实验只针对两阶段方法。实验同时记录补值误差和预测误差，并分析二者之间的相关性。

具体过程如下：

1. 在历史输入中注入缺失；
2. 使用补值方法恢复缺失位置；
3. 计算补值位置上的补值误差；
4. 将补值后的完整序列输入预测模型；
5. 计算未来预测误差；
6. 分析补值误差和预测误差是否同步变化。

若补值误差下降但预测误差不下降，说明补值任务与预测任务并不完全一致。这一分析对应 CRIB 和 CoIFNet 论文中对两阶段方法的批评 [6], [7]。

### 11.3 缺失类型泛化实验

训练阶段只使用随机点缺失，测试阶段分别使用随机点缺失、连续片段缺失、变量通道缺失和混合缺失。

| 训练缺失类型 | 测试缺失类型 |
|---|---|
| 随机点缺失 | 随机点缺失 |
| 随机点缺失 | 连续片段缺失 |
| 随机点缺失 | 变量通道缺失 |
| 随机点缺失 | 混合缺失 |

该实验用于判断模型是否只适应某一种特定缺失模式，还是能够泛化到未见过的缺失类型。

### 11.4 缺失率泛化实验

训练阶段的缺失率从 10% 到 30% 范围内随机采样，测试阶段分别固定为 10% 和 30%。

该实验用于判断模型能否处理不同程度的缺失，并衡量训练阶段缺失率分布对测试阶段不同缺失率的迁移效果。

### 11.5 预测模型复杂度消融实验

该实验比较相同缺失处理方式下，不同预测模型的差异。

| 缺失处理方式 | 对比预测模型 |
|---|---|
| 线性插值 | DLinear、PatchTST、iTransformer |
| SAITS 补值 | DLinear、PatchTST、iTransformer |

该实验用于判断性能提升来自缺失处理方式，还是来自预测模型结构本身。如果在多种填补方式下 DLinear 与复杂模型差距较小，则说明简单趋势建模已经足够；如果复杂模型稳定优于 DLinear，则说明注意力机制或变量依赖建模具有额外价值。

## 12. 评价指标

实验使用四类评价指标。

### 12.1 均方误差

均方误差（MSE）是预测值与真实值差的平方平均值。该指标对较大的预测错误更敏感。

$$
\mathrm{MSE} = \frac{1}{NHC} \sum_{n=1}^{N} \sum_{t=1}^{H} \sum_{c=1}^{C} \left(Y_{n,t,c} - \hat{Y}_{n,t,c}\right)^2
$$

### 12.2 平均绝对误差

平均绝对误差（MAE）是预测值与真实值差的绝对值平均值。该指标更容易解释，反映平均预测偏差大小。

$$
\mathrm{MAE} = \frac{1}{NHC} \sum_{n=1}^{N} \sum_{t=1}^{H} \sum_{c=1}^{C} \left|Y_{n,t,c} - \hat{Y}_{n,t,c}\right|
$$

### 12.3 相对退化率

相对退化率用于衡量模型在缺失条件下相对于完整输入条件的性能下降程度：

$$
\mathrm{Deg}(r) = \frac{\mathrm{MSE}(r) - \mathrm{MSE}(0)}{\mathrm{MSE}(0)}
$$

其中，$\mathrm{MSE}(0)$ 表示无缺失输入下的预测误差，$\mathrm{MSE}(r)$ 表示缺失率为 $r$ 时的预测误差。该值越小，说明模型面对缺失时越稳定。

### 12.4 计算效率

计算效率包括每轮训练时间、单批次推理时间、峰值显存占用和参数量。该指标用于判断方法是否具有实际应用可行性。

## 14. 预期结果

第一，在低缺失率条件下，简单填补后预测与缺失感知方法之间的差距预计较小。当缺失率为 10% 时，历史序列的大部分信息仍然可见，线性插值或基于注意力机制的补值方法能够恢复多数局部结构。

第二，随着缺失率从 10% 上升到 30%，不补值直接预测方法和联合补值预测方法预计退化更平缓。两阶段方法更容易受到补值误差传播影响，而 MissTSM、CRIB 和 CoIFNet 由于显式考虑缺失机制，性能退化预计更小 [5]–[7]。

第三，连续片段缺失预计比随机点缺失更具破坏性。随机点缺失通常仍保留较多邻近观测信息，而连续片段缺失会使某一变量在较长时间内完全不可见，从而破坏时间局部连续性。

第四，补值误差和预测误差预计不完全一致。某些方法可能在补值位置上取得较低误差，但最终预测误差并不最低。这将说明补值任务和预测任务之间存在目标差异。

第五，缺失掩码预计是关键输入信息。在相同填补数值下，带有缺失掩码的模型能够识别哪些位置是真实观测，哪些位置不可信，因此在高缺失率下更稳定。

第六，DLinear 可能在部分数据集上表现出较强竞争力。若 DLinear 在简单填补或 SAITS 补值后接近 PatchTST 和 iTransformer 的表现，则说明该数据集中的主要可预测结构可能较为线性或周期性明显；若复杂模型显著更优，则说明变量依赖和非线性时间模式对缺失条件下的预测更重要。
