3. 训练设置：10 epoch，early stopping patience = 3，batch size = 32，学习率 1e-3。
4. 相对退化率的无缺失基准为对应数据集和预测长度下 3 个 baseline 模型（DLinear / PatchTST / iTransformer）的平均 MSE。
